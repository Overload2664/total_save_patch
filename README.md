## Total Save Patch
A tool to patch total war games to enable them to save at any time in campaign. Currently supported titles are Shogun II, Rome II and Attila.

## Usage
Download the (tool)[https://github.com/Overload2664/total_save_patch/releases/latest/download/total_save_patcher.exe], select your desired game and choose the `empire.retail.dll` file present in the game's main directory.
